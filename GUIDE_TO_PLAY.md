# Chain Reaction Reborn - Guide to Play

Chain Reaction Reborn is a turn-based strategy game where players place orbs on a grid, trigger explosions, capture enemy cells, and create massive chain reactions. The goal is to become the last player with orbs remaining on the board.

## 1. Game Objective

Win by eliminating every other player from the board.

A player is eliminated when they no longer own any cells after the opening phase. When only one player's orbs remain on the board, that player wins.

## 2. Game Modes

### Local Multiplayer

Play with multiple people on the same device.

The game supports 2 to 8 players. Each player takes turns placing orbs. The current player's name and color are shown at the top of the game screen.

### Vs Computer

Play a 1v1 match against the Computer.

Player 1 is human. Player 2 is the Computer. When it is the Computer's turn, the game shows `Computer Thinking...` and temporarily blocks input until the Computer finishes its move.

## 3. Computer Difficulty

In Vs Computer mode, you can choose one of five difficulty levels:

| Difficulty | Computer Behavior |
| --- | --- |
| Easy | Chooses randomly from valid moves. Best for learning the rules. |
| Medium | Prioritizes immediate explosions, safer positions, and basic reinforcement. |
| Hard | Evaluates the board using territory, orb count, captures, and danger. |
| Extreme | Plans deeper by considering the opponent's strongest counter-move. |
| God | The strongest mode, searching further ahead with stronger tactical evaluation. |

## 4. Grid Sizes

Before starting a match, you can choose the board size:

| Option | Size |
| --- | --- |
| X Small | 6 x 4 |
| Small | 8 x 5 |
| Medium | 10 x 6 |
| Large | 12 x 7 |
| X Large | 14 x 8 |

Smaller boards create faster and more explosive matches. Larger boards give players more space to build long-term strategy.

## 5. Placing Orbs

On your turn, tap a valid cell to place 1 orb.

You can place an orb on:

- an empty cell
- a cell already owned by you

You cannot place an orb directly on an opponent's cell.

## 6. Critical Mass and Explosions

Each cell has a different limit depending on its position:

| Cell Position | Critical Mass |
| --- | --- |
| Corner | 2 orbs |
| Edge | 3 orbs |
| Center | 4 orbs |

When a cell reaches critical mass, it explodes. The explosion sends orbs to neighboring cells in four directions: up, right, down, and left.

If an orb from your explosion enters an opponent's cell, that cell is captured and changes to your color.

## 7. Chain Reactions

An explosion can cause nearby cells to reach critical mass and explode as well. This is called a chain reaction.

A strong chain reaction can:

- capture many enemy cells
- eliminate a player from the board
- completely reverse the match
- trigger explosions across the entire grid

The strongest moves are not always the ones that explode immediately. Sometimes a quiet setup move can create a much larger chain reaction later.

## 8. During a Match

The top bar shows:

- the current player's color
- the current player's name
- `Computer Thinking...` when the Computer is choosing a move
- a loading indicator while the game is processing Computer moves or chain reactions

Use the menu button to pause the match.

Use the undo button to return to a previous turn. In Vs Computer mode, undo returns to the human player's turn and skips back past the Computer's latest move.

## 9. Pause Menu

The Pause Menu includes:

- Resume: continue the current match
- New Game: restart with the same setup
- Exit Game: return to the main menu

The game saves unfinished matches automatically. If you leave a match before it ends, you can continue from the saved state later.

## 10. Winner Screen

When a match ends, the Winner screen shows:

- winner name
- total moves
- match duration
- territory percentage controlled by the winner

From this screen, you can:

- Play Again with the same setup
- return to the Main Menu

## 11. Settings

Settings let you customize the game experience.

General options:

- Dark Mode
- Haptic Feedback

Visual and animation options:

- Atom Rotation
- Atom Vibration
- Atom Breathing
- Cell Highlight

Player Settings:

- edit player names
- view each player's assigned color

You can also reset settings to restore the default setup.

## 12. Themes and Shop

The Themes screen lets you change the game's visual palette.

Some themes are available by default. Premium themes can be unlocked through the Shop.

The Shop includes:

- theme packs
- a support development option
- restore purchases

## 13. Strategy Tips

Corner cells are safer and easier to explode because they only need 2 orbs to reach critical mass, but they attack fewer directions.

Edge cells are useful for building controlled chains along the border.

Center cells have the strongest spread because they can explode in four directions, but they require more orbs to activate.

Avoid placing orbs next to an enemy cell that is about to explode unless you are setting a trap.

An immediate explosion is not always the best move. A quiet setup move can create a much larger chain reaction on a later turn.

## 14. Recommended Starting Setup

For new players:

- Mode: Local Multiplayer or Vs Computer
- Computer Difficulty: Easy or Medium
- Grid Size: Small or Medium

For experienced players:

- Mode: Vs Computer
- Computer Difficulty: Hard, Extreme, or God
- Grid Size: Large or X Large
